3
တတိယ - WhatsApp ပို့ခြင်း
💬 WhatsApp Tab မှ ကုန်စည်ရွေးချယ်ပြီး မက်ဆေ့ခ်မျ ကူးယူ WhatsApp မှာ ပို့ပါ
💡 အကြံဉာဏ်များ
✓ ကုန်စည်အီမိုဂျီ ကောင်းကောင်း ရွေးချယ်ပါ - Customer ဖက်ကြည့်မှု ကောင်း
✓ မက်ဆေ့ခ်မျများ အများစုလို ပြင်ဆင်ပါ - ကုန်သည်ဖက်ကုံးမှုအလျောက်
✓ အသုံးခြင်း မှတ်တမ်းတင်ပါ - ကုန်စည်လည်ပတ် သိမ်းသိမ်း ကြည့်နိုင်
✓ အတည်းအနေ အဆင့်အတန်း ပြင်ဆင်ပါ - ကုန်သည်အချက်အလက် အဆင့်အတန်း
🔗 GitHub Pages မှာ Deploy လုပ်ရန်
1. GitHub မှ Repository ဖန်ဆင်းရန်
သင့်ရဲ့ GitHub repository မှ "index.html" အဖြစ် ဒီ file ကို upload လုပ်ပါ

2. GitHub Pages activate လုပ်ရန်
Settings → Pages → Deploy from branch (main) ရွေးချယ်ပါ

3. URL ကူးယူ ရန်
https://yourusername.github.io/repository-name/

4. WhatsApp မှာ အသုံးပြုရန်
Customer မေးတိုင်းအတွက် URL ကို အထက်မှ copy လုပ်ပြီး WhatsApp မှာ ပို့ပါ
